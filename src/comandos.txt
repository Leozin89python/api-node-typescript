
comando 1 (arqv de confg ts)
npx tsc --init

comando 2 (comando de compilação)
npx tsc

comando 3 (execução do arquivo)
node nome-pasta-criada/nome-do-arquivo.js
EX:
node dist/server.js

comando 4 (rodando como dev)
npm run dev

node server.js

o comando 'node' são o path(caminho),
